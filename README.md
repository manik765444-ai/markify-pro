# markify-pro
================

A Node Markdown Blog that weaves code and prose into a unique dig.

## Description
---------------

markify-pro is a minimalist blog that lets you write articles in Markdown and publishes them as a simple, easy-to-read blog. It's the perfect space for developers who want to share their knowledge, showcase their projects, and document their journey.

## Features
------------

* **Markdown Support**: Write articles in Markdown and see them rendered as beautiful, readable text.
* **Code Highlighting**: Syntax highlighting for code snippets, because who doesn't love pretty code?
* **Customizable**: Make markify-pro your own by tweaking the layout, adding custom CSS, and modifying the template.
* **Simple**: No fuss, no muss. Just write, save, and publish.

## Installation and Usage
-------------------------

1. **Install dependencies**: `npm install`
2. **Run**: `node app.js`
3. **Access**: <http://localhost:3000>

## Technologies
--------------

* **Node.js**: The power behind markify-pro, providing a lightweight and fast server.
